PDX Flight Entry — iPhone Web App

How to use on iPhone:
1. Upload the ZIP contents to any HTTPS web host, such as GitHub Pages, Netlify, or an internal company web server.
2. Open index.html in Safari using the hosted HTTPS link.
3. Tap Share, then Add to Home Screen.
4. Open PDX Flights from the Home Screen.

Important:
- Do not rely on opening index.html directly from the iPhone Files app. iOS may block or limit JavaScript for local files, which can make buttons/dropdowns fail.
- Data saves locally on the device/browser using localStorage.
- Export JSON or CSV for backup.
- Private Browsing may prevent persistent storage.
